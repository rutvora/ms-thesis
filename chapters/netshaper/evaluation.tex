\section{Evaluation}
\label{sec:netshaper-evaluation}

In our evaluation, we answer the following questions:
(i) Does NetShaper limit the bandwidth utilisation of the underlying network link? (see \Cref{subsec:netshaper-evaluation-bw})
(ii) How many packets can NetShaper process per second? (see \Cref{subsec:netshaper-evaluation-num-packets})
(iii) Does NetShaper throttle the end host application? (see \Cref{subsec:netshaper-evaluation-http-reqs})
(iv) What is the latency incurred by NetShaper (in relation to window $W$)? (see \Cref{subsec:netshaper-evaluation-latency})
(v) How many parallel clients can NetShaper support? (see \Cref{subsec:netshaper-evaluation-num-clients})
